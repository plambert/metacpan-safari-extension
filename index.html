The MetaCPAN Safari Extension

An extension to automatically redirect *.cpan.org pages to *.mcpan.org.

Download

Why?

I'm tired of clicking the MetaCPAN link on the top right of CPAN pages, and I prefer the MetaCPAN layout.

How do I use it?

Download the extension here: https://plambert.net/metacpan-extension/MetaCPAN.safariextz

Double-click it to install it.

Frequently Asked Questions

How can I be sure this isn't stealing my data, or otherwise nefarious in some way?

Feel free to check out the source code from GitHub and build it yourself.
